# pytis

파이썬 콘솔 테트리스 - 2022 수행평가

## 제약사항

- 외부 라이브러리 사용 금지
- 파이썬 변수, 함수, 반복문, 조건문만 사용 가능
- 콘솔 출력만 사용 가능
- 단일 파일로 작성

## 구현 내용

- 콘솔에서 렌더링하기 위한 단순한 게임 엔진구현
- ANSI 컬러 및 UNIX 컬러 코드를 사용하여 콘솔에 색상 출력
- 점수에 따라 중력 증가
- hold, hard drop, soft drop, move, rotate, clear, game over 등 기본적인 테트리스 기능 구현
- 최고 점수 기록
